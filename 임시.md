## 📚 MCAT (Multimodal Co-Attention Transformer)

> **Paper:** [Multimodal Co-Attention Transformer for Survival Prediction in Gigapixel Whole Slide Images](https://arxiv.org/abs/2103.04814) (ICCV 2021)  
> **Official Repo:** [mahmoodlab/MCAT](https://github.com/mahmoodlab/MCAT) (이미지 처리: [mahmoodlab/CLAM](https://github.com/mahmoodlab/CLAM))

### 1. Research Overview
이 연구는 **기가픽셀 규모의 WSI(병리 이미지)**와 **유전체 데이터**를 결합하여 환자의 생존율을 예측하는 멀티모달 프레임워크를 제안합니다.

* **Key Challenge:** 대용량 이미지 처리의 계산 복잡성과 이미지-유전자 데이터 간의 이질성 해결.
* **Solution:** 유전체 정보를 가이드로 삼아 이미지의 핵심 부위를 찾는 **Co-Attention** 메커니즘 도입.

### 2. Data
연구팀은 대규모 공공 암 데이터 컨소시엄인 **TCGA(The Cancer Genome Atlas)**의 5가지 암종 데이터 사용.
- **암종 종류**: 방광암(BLCA), 유방암(BRCA), 교모세포종 및 저등급 신경교종(GBMLGG), 폐선암(LUAD), 자궁내막암(UCEC).
- **데이터 규모**: 총 **4,730개의 WSI**와 약 **6,700만 개의 이미지 패치**를 포함하는 데이터 처리.
- **유전체 데이터**: 유전자 변이 상태, 복제수 변이(CNV), RNA-Seq 발현량 등 활용.
- **Architecture:** `model_coattn.py`의 `GenomicGuidedCoAttention` 클래스가 핵심 두뇌 역할 수행.

### 3. Methodological Pipeline
본 프로젝트(REDCAB)에서 차용한 핵심 모델 구조입니다.

| Step | Component | Description |
| :--- | :--- | :--- |
| **Step 1** | **Feature Extraction** | ResNet50 등을 이용해 WSI를 패치 단위의 특징 벡터($H_{bag}$)로 변환 |
| **Step 2** | **Genomic Embedding** | 6가지 생물학적 범주(Pathways)에 따라 유전체 데이터를 $G_{bag}$으로 구성 |
| **Step 3** | **Co-Attention** | 유전체 데이터를 Query로 사용하여 이미지 패치 중 중요한 부분을 필터링 |
| **Step 4** | **Survival Prediction** | 융합된 벡터를 통해 생존 확률(S-NN) 및 위험도 예측 |

### 4. Result
- **성능 우위**: 기존의 최첨단(SOTA) 방법론들(DeepAttnMISL, Attention MIL 등)보다 전체적으로 **3.0%에서 6.87% 향상된 예측 정확도(c-Index)** 보임.
  - SOTA (State-of-the-Art): 특정 분야에서 현재까지 알려진 기술 중 **가장 높은 수준의 성능을 내는 모델이나 방법**을 의미
  - **조직 이미지(WSI)만 쓰는 분야의 챔피언**: `DeepAttnMISL` (이미지 분석 전문 모델) // **기본적인 다중 인스턴스 학습(MIL)의 표준**: `Attention MIL` (가장 널리 쓰이는 강력한 기본 모델).
  - 현재 '대장암 특화 이미지-멀티오믹스 추론'과 관련된 SOTA는 **MOMA**임.
- **해석 가능성**: 연구팀은 **공동 주의 집중(Co-attention)** 가중치를 시각화하여, 특정 유전체 범주(예: 종양 유발 유전자)가 실제 이미지의 어떤 병리적 영역(예: 침습적 종양 세포)과 연관되는지 열지도(Heatmap)로 시각화함.
  - Co-Attention:
    - **이미지가 유전체를 보고**, 동시에 **유전체가 이미지를 봄.**
    - **작동 원리**: "이 구역의 암세포 모양이 수상한데, 유전체 데이터에서 관련 변이가 있는지 찾아봐!" 혹은 "이 유전자에 변이가 있으니까 이미지에서 이 부분을 더 집중해서 봐!"라고 실시간으로 정보 교환.
    - **연구에의 의미**: 이 방식은 **"유전적 근거를 가진 시각적 예측"**이라는 목표에 가장 완벽하게 부합하는 기술.
