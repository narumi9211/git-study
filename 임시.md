| Step | Component | Description |
| :--- | :--- | :--- |
| **Step 1** | **Feature Extraction** | ResNet50 등을 이용해 WSI를 패치 단위의 특징 벡터($H_{bag}$)로 변환 |
| **Step 2** | **Genomic Embedding** | 6가지 생물학적 범주(Pathways)에 따라 유전체 데이터를 $G_{bag}$으로 구성 |
| **Step 3** | **Co-Attention** | 유전체 데이터를 Query로 사용하여 이미지 패치 중 중요한 부분을 필터링 |
| **Step 4** | **Survival Prediction** | 융합된 벡터를 통해 생존 확률(S-NN) 및 위험도 예측 |
